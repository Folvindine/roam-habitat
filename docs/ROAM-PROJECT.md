# ROAM Project Knowledge Base

## Project Identity
- **Full name**: Robotic Off-grid Autonomous Mover
- **Domain**: roamhabitat.com
- **Phase**: Concept & AI Navigation Development
- **Approach**: Software-first (simulation before hardware)

## Core Thesis
Global demographics are collapsing. Road infrastructure built for 8 billion won't be maintained for 6 billion. We need vehicles that don't depend on roads. ROAM is a hexapod walking habitat — terrain-independent, self-sufficient, AI-guided.

## Why Hexapod?
- **Static stability**: Alternating tripod gait keeps 3 legs grounded (stable triangle)
- **Fault tolerance**: Lose a leg → switch to pentapod gait and walk home safely
- **Speed optimal**: 6 legs is the speed optimum for legged locomotion (more legs = diminishing returns)
- **Multi-function**: Spare legs serve as manipulators (anchoring, lifting, stabilizing)

## Core Systems
1. **Locomotion**: 6 legs, 18+ DOF, adaptive gait patterns
2. **Navigation**: LiDAR + satellite terrain data + neural terrain classification
3. **Power**: Solar array + hydrogen fuel cell + regenerative leg hydraulics
4. **Habitat**: Easy-clean interiors, 3D-printable modules, closed-loop water
5. **Communications**: Starlink + mesh networking + emergency beacon
