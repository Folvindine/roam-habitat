# ROAM — Robotic Off-grid Autonomous Mover

A terrain-independent hexapod mobile habitat. No roads. No wheels. No infrastructure.

Wake up in the Amazon. Walk to the mountains. Live anywhere on Earth.

## What is ROAM?

ROAM is a self-sufficient walking dwelling built on a hexapod (six-legged) platform. It traverses any terrain — dense forest, mountain passes, river crossings, desert, snow — carrying a fully habitable living module powered by solar and hydrogen energy, guided by terrain-aware AI.

## Why?

Global demographics are collapsing. Birth rates are falling below replacement in most developed nations. The road infrastructure built for 8 billion people won't be maintained for 6 billion. We need vehicles that don't depend on roads.

## Architecture

ROAM is integrated into the [AION](https://github.com/folvin/aion) ecosystem:
- **AION** — Central brain, memory, and coordination
- **Pilgrim** — R&D intelligence scout (researches ROAM topics daily)
- **Laila** — Personal AI assistant (captures ROAM notes via Telegram)

## Core Systems

- **Locomotion:** Hexapod adaptive gait (6 legs, 18+ DOF)
- **Navigation:** Terrain-aware AI (LiDAR, satellite data, neural terrain classification)
- **Power:** Solar array + hydrogen fuel cell + regenerative leg hydraulics
- **Habitat:** Easy-clean, 3D-printable modular interior, closed-loop water
- **Communications:** Starlink + mesh networking + emergency beacon
